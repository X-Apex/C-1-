# ⚙️ C++

En esta sección encontrarás los **mismos ejercicios** que ya fueron trabajados en **Java** y **Python**.  

La diferencia está únicamente en el **lenguaje de programación**, ya que aquí se presentan las soluciones en **C++**.  

El objetivo es mostrar cómo los mismos problemas pueden resolverse en distintos lenguajes, reforzando la lógica y la capacidad de adaptación.  

De esta manera, podrás comparar la sintaxis y las particularidades de cada lenguaje, mientras practicas con ejemplos concretos.
